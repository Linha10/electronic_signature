# 電子簽名 electronic_signature

使用smooth-signature & socket.io 搭配element.ui製作的簽名。
可用QRcode掃描開啟手機版本(目前僅可在電腦網頁上操作)。
使用者id為hard code。

##  Dialog式QRcode:

- 在QRcode圖片放入下方的空白格內，會產生QRcode對應的資料
